Common Array Methods

concat
- Joins multiple arrays
join
- joins all the elements together into a string
toString
- Returns the array as a string
indexOf
- Searches for a specific element
lastIndexOf
- Returns the last element that meet the critera
slice
- Returns new array from the specified index and length
sort
- sort alpha or by desired function
splice
- adds or deletes specified index(es) to or from the array
