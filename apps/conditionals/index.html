If it evaluates to false then the statement after the colon is envoked. these can be stacked by placing the next conditonal after the colon followed by another quesion mark and colon

The switch Statement

The switch statemen checks for different cases of possible variable value and runs the according statements. note the break statement after each case, if this is ommited the rest of the cases will be evaluated as true. so to go to the end of the switch statement after the case use the break


//Basic switch statement evaluating color
switch(color){
  case 'red':
    message = 'Stop!';
    break;
  case "yellow":
    message = 'Proceed with caution';
    break;
  case 'green':
    message = 'go ahead';
    break;
  default:
    message = 'The light is out, use caution';
}
