Getters and Setters

If get and set are defined within an object they are called in the background whenever a property is accessed with the dot notation

Take note of the underscore, this is necessary to prevent JavaScript from going into an infinite loop


class Person {
  constructor(name){
    this.name = name;
  }

  setName(newName) {
    this.name = newName;
  }

  get name(){
    return this._name;
  }

  set name(n){
    //set the first char to uppercase
    // the rest to lowercase
    n = n.charAt(0).toUpperCase() + n.slice(1).toLowerCase();
    this._name = n;
  }

}

let Mary = new Person('mary');
Mary.setName('pAttY');
console.log(Mary.name); // logs "Patty" to the console
